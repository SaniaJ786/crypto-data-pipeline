# Automated Crypto Data Pipeline

## Overview
A Python + SQL pipeline that automatically fetches crypto prices every 5 minutes and stores them in a database.

## Problem
Manual data collection is slow and error-prone.

## Solution
This pipeline:
- Calls crypto APIs using Python  
- Cleans & transforms data  
- Inserts into SQL tables  
- Runs automatically via Task Scheduler

## Tools
- Python  
- SQL  
- API  
- Windows Task Scheduler

## Output
Always fresh data ready for dashboards
